❓ Bantuan & FAQ

Bagaimana cara booking kos?
Pilih kos yang diinginkan → klik Booking Sekarang → isi data → pilih kamar & durasi → lanjut pembayaran.
›
Bagaimana cara menghubungi pemilik kos?
Buka halaman Detail Kos → klik tombol Chat untuk memulai percakapan langsung dengan pemilik.
›
Apakah data saya aman?
Ya, data Anda tersimpan dengan aman menggunakan enkripsi dan sistem keamanan Supabase.
›
Bagaimana cara mendaftarkan kos saya?
Daftar sebagai Pemilik Kos → masuk ke Dashboard → Kelola Kos → Tambah Kos Baru → isi data lengkap → Simpan.
›
Bagaimana jika ada masalah pembayaran?
Hubungi kami melalui email support atau WhatsApp yang tersedia di bagian Hubungi Kami di bawah.
